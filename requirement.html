Requirement

Requirement: Choose any object in the real world, such as a class, subject, lecturer, etc. Then design the function to manage that object on HTML including the features of adding, editing, deleting, and searching for that object. For example, make the function of managing students, including the features of adding, editing, deleting, and searching.

Note: Group members cannot choose the same objects.
